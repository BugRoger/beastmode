# Release: retro-report-format

**Version:** v0.14.26
**Date:** 2026-03-08

## Highlights

Retro reports now show actual content being changed instead of opaque counts and one-liners. Gate-aligned layout with visual prefixes (~, +, >>, ^) makes approval decisions scannable.

## Features

- Redesigned context changes template (step 4) with edit/create prefixes and actual content bullets
- Redesigned meta review summary (step 7) to show L2 edits inline instead of count-heavy block
- Redesigned records template (step 8) with append/create prefixes and one-sentence summaries
- Redesigned promotions template (step 9) showing actual ALWAYS/NEVER rules being promoted
